# Terraformのチュートリアル

https://learn.hashicorp.com/collections/terraform/aws-get-started

# コマンド集
```
// terraformの初期化
$ terraform init

// terraformの適応（実行）
$ terraform apply

// 状態の確認
$ terraform show

// リソースの終了
$ terraform destroy

// 出力データの照合
$ terraform output

// terraform Cloudにログインする
$ terraform login

// 状態の一覧
$ terraform state list

// .tfファイルのフォーマット
$ terraform fmt

// .tfファイルのバリデーション
$ terraform validate
```
